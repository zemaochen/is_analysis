# 接口：getSemester  [返回](../README.md)
用例： [选择学期](../用例/选择学期.md)

- 功能：
    取得历史学期
- 权限：
    学生/老师已经登录。    
- API请求地址： 
    接口基本地址/v1/api/getSemester

- 请求方式 ：
    POST

- 请求实例：
```json
{ 
   "user_id": "123456"
}
 ```
        
- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |user_id|已经登录用户的user_id值，对应表user的user_id|
  
- 返回实例：
```json
{ 
     "status": true,
     "info": null,
     "data":[
      {
        "semester": "xx学年第一学期"
      },
      {
        ...其他学期
      }
     ]
}
 ```
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
  |data|返回学期数组信息|
